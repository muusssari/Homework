# Sever for homework app

- Testing made using Jest and using simple data validation for notes.

## Swagger

I made endpoint to create, get and delete. Two different delete endpoints for deleting one or deleting array of notes.

## Jest

Testing Endpoints with Jest

![Jest](./images/JestTests.png)


## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
